**Smart Assistive Glasses**



An AI-powered wearable device for visually impaired individuals that provides real-time object detection and obstacle awareness using Raspberry Pi Zero 2 W and TensorFlow Lite.



**Features**



\- Real-time object detection

\- Obstacle detection using HC-SR04

\- Audio feedback

\- Push button controls

\- Lightweight AI model for embedded devices



**Hardware Components**



\- Raspberry Pi Zero 2 W

\- Pi Camera

\- HC-SR04 Ultrasonic Sensor

\- Earphones

\- Push Buttons



**Software Stack**



\- Python

\- OpenCV

\- TensorFlow Lite

\- RPi.GPIO

\- Pygame
